# 🌱 Plantive Web Prototype

A **mobile-first web prototype** for the **Plantive Ecosystem** — an AI-powered smart plant care platform.

This web application simulates the behavior and layout of a **mobile application** and is designed as a **prototype before building the final Android app**.

The goal is to quickly test UI, workflows, and backend integration before moving to full mobile development.

---

# 📱 Features

### 🌿 Garden Management

* View garden overview
* Track plant health
* Monitor plant statistics

### 🌤 Weather Information

* Temperature
* Humidity
* Weather conditions

### 🤖 AI Plant Assistant

* Ask plant care questions
* Future integration with AI diagnosis

### 📅 Daily Tasks

* Plant watering reminders
* Care schedule

### 🛒 Marketplace (Prototype)

* Browse plant products
* Future e-commerce integration

### 👤 User Profile

* User information
* Account management

---

# 🎯 Purpose

This project serves as a **rapid prototype** for the Plantive mobile application.

Benefits of building this web prototype first:

* Test UI/UX quickly
* Validate features
* Integrate backend APIs
* Reduce mobile development complexity
* Iterate faster

After this prototype is validated, the UI and logic will be migrated to the **mobile application**.

```bash

```

### 3. Open the app

Simply open:


in your browser.
